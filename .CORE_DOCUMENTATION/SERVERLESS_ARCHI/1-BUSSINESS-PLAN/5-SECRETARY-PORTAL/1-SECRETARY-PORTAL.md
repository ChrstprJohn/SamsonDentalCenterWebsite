# Secretary Portal Documentation Index

The Secretary Portal is the workspace for clinic staff to manage appointment requests, book walk-ins/phone calls, handle patient check-in/check-out tracking, finalize invoices, and view logs.

## Navigation & Page Structure

We break down the secretary portal into the following sections. Each has its own dedicated page file for step-by-step implementation:
